ABOUT

Trabalho realizado por: Martim Calisto Pinheiro

Para a UC Introdução à Computação Gráfica.


Este projeto é uma aplicação web interativa que simula o corte de árvores em um ambiente 3D, utilizando tecnologias modernas de renderização e física.

Descrição: Aplicação web 3D interativa com simulação física de árvores usando Three.js e cannon-es, otimizada para execução no navegador via GitHub Pages.

Tecnologias usadas:

- Three.js (renderização 3D)
- cannon-es (motor físico)
- glTF (modelos 3D)

Elementos gratuitos e créditos:

- Modelos 3D de árvores retirados da plataforma PolyHaven com licença gratuita.
- Texturas e sons usados são livres de direitos ou sob licenças Creative Commons.
